## 1. 动机与问题诊断

### 1.1 当前架构 (v4.4) 的核心问题

```
GaborFilterBank (32 filters, 一次性响应图)
  ├─ encode()         → V1  (4×4 网格池化, 96d)    ← 所有滤波器响应混在一起
  ├─ encode_v2()      → V2  (2×2 网格池化, 64d)    ← 同一组响应的粗池化
  ├─ encode_v4()      → V4  (1×1 全局池化, 64d)    ← 同一组响应的更粗池化
  ├─ encode_color()   → Color (RG/BY, 42d)
  ├─ encode_pulvinar()→ 低空间频率捷径 (32d)
  └─ encode_dorsal()  → 背侧梯度+质心 (32d)
```

三个问题：

1. **V1/V2/V4 不是独立加工** — 它们只是同一 Gabor 响应图的不同空间分辨率下采样。生物视觉中 V2 和 V4 有 V1 没有的特征检测能力（纹理综合、曲率、颜色恒常性）。
2. **M/P/K 通路从未分离** — 从视网膜的三种神经节细胞到 LGN 的六层到 V1 的层状结构到 V2 的三类条纹，三条通路始终保持解剖学分离。当前代码把所有 Gabor 响应混在一起池化。
3. **没有层级间交互** — 生物视觉中"除视网膜→LGN 外所有连接都是双向的"。当前代码只有前馈编码，没有反馈预测，预测编码的核心机制没有进入视觉处理。

### 1.2 设计目标

- 建立 M/P/K 三流分离的并行通路骨架
- LGN 从被动编码器升级为主动门控中继站
- 每个皮层视觉区成为独立模块，含前馈 + 反馈预测 + 局部 Hebb 可塑性
- 格式塔闭合律、共同命运律、简洁律从层级动力学中涌现（非手工特征）
- 上丘→丘脑枕第二条通路独立于 LGN 通路
- 不违背项目三条方法论承诺（仿脑区结构、无现象学、无 LLM）

## 2. 架构总览

### 2.1 三条并行通路

| 属性 | M 通路 (运动/空间) | P 通路 (形状/细节) | K 通路 (颜色) |
|------|-------------------|-------------------|--------------|
| **视网膜细胞** | Parasol (Pα) | Midget (Pβ) | Bistratified |
| **传导速度** | 快 (~20 m/s) | 慢 (~10 m/s) | 中等 |
| **对比灵敏度** | 高 (检测 ~2% 对比度) | 低 (需 ~10% 对比度) | 中 |
| **空间分辨率** | 低 | 高 | 中 |
| **颜色** | 无 | 红-绿拮抗 | 蓝-黄拮抗 |
| **Gabor 参数** | 大 σ, 低空间频率, 高时间频率 | 小 σ, 高空间频率 | 中 σ, 中空间频率 |
| **V1 入口** | 4Cα → 4B | 4Cβ → 2/3 斑点间区 | 2/3 斑块 |
| **V2 出口** | 粗条纹 (Thick) | 苍白条纹 (Pale) | 细条纹 (Thin) |
| **最终通路** | 背侧: MT → MST → PPC | 腹侧: V4 → IT | 腹侧: V4 → IT |

### 2.2 全系统数据流

```
视网膜 (photoreceptors → ganglion cells)
  ├─ M 型 (parasol):  低分辨率, 快速, 无颜色
  ├─ P 型 (midget):   高分辨率, 慢速, 红-绿拮抗
  └─ K 型 (bistratified): 中等分辨率, 蓝-黄拮抗
       ↓
LGN (6层 + 层间, 主动门控)
  ├─ 大细胞层 1-2:  M 通路中继 + V1-L6 反馈增益 + 脑干状态调制
  ├─ 小细胞层 3-6:  P 通路中继 + V1-L6 反馈增益 + 脑干状态调制
  └─ 粒状细胞层间区: K 通路中继 + 上丘输入
       ↓
V1 (层状结构, M/P/K 通路保持分离)
  ├─ 4Cα (M入口):    Gabor 运动/方向滤波 → 4B 层 (M出口)
  ├─ 4Cβ (P入口):    Gabor 空间频率/朝向滤波 → 2/3 斑点间区 (P出口)
  ├─ 2/3 斑块 (K入口): 颜色拮抗处理 → V2 细条纹 (K出口)
  ├─ 第5层:          → 上丘 (眼球运动控制)
  └─ 第6层:          → LGN 反馈 (皮层-膝状体门控调节)
       ↓
V2 (三类条纹, 并行延续)
  ├─ 粗条纹 (M→):  方向池化 + 运动对比 → MT
  ├─ 苍白条纹 (P→): 空间频率 + 朝向交互 → V4
  └─ 细条纹 (K→):  颜色恒常性初步 → V4
       ↓
┌── 背侧通路 ("Where/How") ────┐  ┌── 腹侧通路 ("What") ──────────┐
│ MT (中颞区/V5):              │  │ V4: 曲率检测 + 颜色恒常性       │
│  → 方向选择性柱状组织         │  │  → M/P/K 在此初步汇合            │
│  → 运动能量 + 方向检测        │  │ IT (PIT→CIT→AIT):               │
│  → 接受 V1-4B + V2-粗条纹输入 │  │  → Hebb 物体类别学习             │
│ MST (内上颞区):               │  │  → 位置/大小不变性              │
│  → 光流模式 (扩张/旋转/平移)   │  │  → 面孔选择性 (FFA同源)         │
│ PPC (后顶叶):                 │  └────────────────────────────────┘
│  → 视觉→行动转换              │
└───────────────────────────────┘

第二条通路 (膝状体外):
  视网膜 → 上丘 (SC) → 丘脑枕 (Pulvinar) → 纹外皮层 (V2/V4/MT)
  → 快速空间定位 + 眼动协调 + 显著性检测 + "盲视"通路
```

### 2.3 双向连接 (前馈 + 反馈)

除视网膜→LGN 以外，**所有层级节点之间都是双向连接**：

```
前馈 (自下而上):
  视网膜 → LGN → V1 → V2 → V4 → IT         (快速, 粗糙)
              ↘ V1-4B → MT → MST → PPC     (背侧通路)

反馈 (自上而下):
  IT → V4 → V2 → V1 → LGN                   (慢速, 携带预测)
         MST → MT → V1-4B                    (背侧反馈)
```

反馈的功能:
- **V1→LGN**: 调节 LGN 增益，增强或抑制特定空间/时间频率的传递
- **V2→V1**: 携带 V2 整合后的"轮廓预期"→ V1 简单细胞的朝向偏好可以被上层预期偏置
- **V4→V2**: 携带颜色恒常性和曲率预期
- **IT→V4**: 携带物体级别的预测——"如果这是苹果，V4 应该看到红色曲面"
- **MST→MT→V1**: 携带运动连贯性预期——"这些点应该一起向右移"

## 3. 模块设计

### 3.1 统一接口模式

每个皮层视觉区遵循相同的接口协议（不强制继承，但概念一致）：

```
class VisualArea:
    def feedforward(lower_output) → representation   # 自下而上: 特征编码
    def feedback(higher_prediction) → prediction      # 自上而下: 生成预测
    def prediction_error() → PE                        # PE = |repr - prediction|
    def hebb_update() → None                           # 局部 Hebb 可塑性
    def set_gain(gain_vector) → None                   # FPN 探照灯增益调制
```

### 3.2 视网膜 (Retina) — 神经节细胞分型输出

**文件**: `cerebrum/occipital_lobe/retina_lgn.py` (重写)

**职责**: 将输入图像转换为三种神经节细胞类型的响应

```
输入: 图像 (H × W × 3 RGB)
输出: {
    "M":  parasol_responses  (低分辨率, 灰度, 高对比灵敏度),
    "P":  midget_responses   (高分辨率, 红-绿拮抗),
    "K":  bistratified_responses (中等分辨率, 蓝-黄拮抗),
}
```

**计算**:
- M 型: 高斯模糊 (大 σ 模拟低空间分辨率) → 灰度 → Gabor 低频核 (2-4 scales, 低空间频率)
- P 型: 原始分辨率 → R-G 拮抗通道 → Gabor 高频核 (4 scales, 高空间频率)
- K 型: 降采样 (中等) → B-Y 拮抗通道 → Gabor 中频核

**关键参数**: Gabor 尺度范围、拮抗通道权重

### 3.3 LGN — 6 层主动门控

**文件**: `cerebrum/thalamus/lgn.py` (新建)

**职责**: 中继视网膜输出到 V1，同时受 V1 反馈和脑干状态的主动门控

```
LGN 六层结构:
  第 1 层 (大细胞, 对侧眼): M 通路
  第 2 层 (大细胞, 同侧眼): M 通路
  第 3 层 (小细胞, 同侧眼): P 通路
  第 4 层 (小细胞, 对侧眼): P 通路
  第 5 层 (小细胞, 同侧眼): P 通路
  第 6 层 (小细胞, 对侧眼): P 通路
  层间区 (粒状细胞):        K 通路
```

**门控机制**:
1. **V1 第 6 层反馈** → 调制各层增益: `gain_layer = 1.0 + α × feedback_from_V1_L6[layer]`
2. **脑干状态调制** (来自蓝斑核 NE / 中缝核 5-HT / 脑桥胆碱能):
   - 清醒/注意状态 → **中继模式** (tonic mode): 线性传递
   - 睡眠/低唤醒状态 → **爆发模式** (burst mode): 阈值升高, 传递效率降低
3. **TRN 侧抑制**: `output[layer] = output[layer] - β × sum(neighboring_layers)`

**输出**: `{M: gated_M, P: gated_P, K: gated_K}` (门控后的三种通道信号)

**互操作性**: 脑干状态参数从 `Agent.brainstem_state` 读取（唤醒度来自扣带回的 arousal 值）

### 3.4 V1 — 层状结构 + 通路分离

**文件**: `cerebrum/occipital_lobe/v1.py` (重写, 从薄封装升级为完整模块)

**职责**: 第一皮层加工站, 保持 M/P/K 通路分离, 不做跨通路融合

```
V1 内部层次:
  4Cα: M 通路入口 → 运动/方向选择 Gabor
  4Cβ: P 通路入口 → 空间频率/朝向选择 Gabor
  2/3 斑块 (blobs):    K 通路入口 + 颜色拮抗
  2/3 斑点间区 (interblobs): P 通路的朝向选择性
  4B: M 通路出口 → MT + V2 粗条纹
  5:  输出到上丘 (眼球运动)
  6:  反馈到 LGN
```

**前馈处理**:
- `M_V1 = Gabor_motion(LGN.M)` — 瞬态/运动选择性 Gabor (奇数对称核)
- `P_V1 = Gabor_orientation(LGN.P)` — 静态朝向选择 Gabor (偶数对称核)
- `K_V1 = Color_opponency(LGN.K)` — 颜色拮抗 + 低通

**反馈处理**:
- `feedback_to_LGN = W_v1_to_lgn × PE_V1` — 将 V1 的预测误差映射为 LGN 增益调制信号
- `feedback_from_V2 = ...` — 接收 V2 的预测, 调制 4C 层的感受野增益

**预测误差**:
- `PE_M_V1 = |M_V1 - pred_from_V2_thick|` (M 通道误差)
- `PE_P_V1 = |P_V1 - pred_from_V2_pale|` (P 通道误差)
- `PE_K_V1 = |K_V1 - pred_from_V2_thin|` (K 通道误差)

**Hebb 增益** (保留现有机制): 高频激活的特征获得增益提升

### 3.5 V2 — 三类条纹

**文件**: `cerebrum/occipital_lobe/v2.py` (重写)

**职责**: 三类条纹并行处理, 向 MT 和 V4 分叉

```
粗条纹 (Thick, M→):  方向池化 → 运动对比度 → 向 MT 前馈
苍白条纹 (Pale, P→):  空间频率交互 + 共线促进 → 向 V4 前馈
细条纹 (Thin, K→):   颜色恒常性初步 → 向 V4 前馈
```

**条纹间横向连接** (关键创新):
- 粗条纹 → 苍白条纹: "同一空间位置的 M 和 P 通道信息可能属于同一物体" → 共同命运律的基础
- 细条纹 → 苍白条纹: 颜色边界往往是物体边界 → 边界归属信号

**反馈**:
- 从 MT 接收: 运动预期 (调制粗条纹增益)
- 从 V4 接收: 形状和颜色预期 (调制苍白和细条纹增益)
- 向下: 向 V1 发送预测 (按条纹映射到 V1 对应层)

### 3.6 MT (V5) — 运动检测

**文件**: `cerebrum/temporal_lobe/mt_cortex.py` (新建)

**职责**: 方向选择性柱状组织, 运动能量计算

**前馈**:
- 接受 V1-4B (M 输出) 和 V2 粗条纹
- 方向选择性: 8 个方向通道, 每个通道计算该方向的运动能量
- 运动对比度: 当前帧 vs 前一帧的方向能量差异

**反馈**:
- 从 MST: 光流连贯性预期 → "这些点应该一起向右移"
- 向 V2 粗条纹: 运动预测 → "同一方向运动的特征应归为一组" (共同命运律的关键)

**预测误差**: 方向 × 空间位置的 PE 矩阵

### 3.7 MST — 光流模式

**文件**: `cerebrum/temporal_lobe/mst_cortex.py` (新建)

**职责**: 检测整体运动模式 (扩张/收缩/旋转/平移)

**前馈**: 从 MT 的方向能量图池化 → 光流模板匹配
- 扩张检测: 径向向外的方向模式
- 收缩检测: 径向向内的方向模式
- 旋转检测: 切向方向模式
- 平移检测: 均匀方向模式

**反馈**: 向 MT 发送场景级运动预期

### 3.8 V4 — 曲率 + 颜色恒常性

**文件**: `cerebrum/occipital_lobe/v4.py` (重写)

**职责**: M/P/K 初步汇合点, 曲率检测, 颜色恒常性

**前馈**:
- 从 V2 苍白条纹 (P): 形状信息 → 曲率检测 (二阶方向导数)
- 从 V2 细条纹 (K): 颜色信息 → 颜色恒常性计算 (全局平均色补偿)

**反馈**:
- 从 IT: 物体预测 → "预期看到一个红色曲面"
- 向 V2: 形状和颜色预期

**汇合**: `V4_rep = concat[shape_P, color_K] × cross_channel_interaction` (M/P/K 在此首次交互)

### 3.9 IT 皮层 — 物体表征

**文件**: `cerebrum/temporal_lobe/it_cortex.py` (填充实现)

**职责**: 腹侧通路终端, Hebb 物体类别学习, 位置/大小不变性

**前馈**: 从 V4 接收 曲率 + 颜色融合表征

**处理**:
1. **空间池化** → 位置不变性 (感受野覆盖整个视野)
2. **尺度池化** → 大小不变性
3. **Hebb 簇激活** → 物体类别 (不预设标签, 由共现统计形成簇)
4. **面孔选择性** (未来): 特定 Hebb 簇自然调谐到高频出现的高复杂度模式

**反馈**: 向 V4 发送物体预测 → "如果当前物体是 X, V4 应该看到特征 Y"

### 3.10 上丘 (SC) — 快速空间定向

**文件**: `brainstem_cerebellum/midbrain/superior_colliculus.py` (新建)

**职责**: 显著性图, 空间定向, 眼动协调

**前馈**:
- 直接接受视网膜输入 (不经 LGN) → 快速处理
- 显著性计算: 空间对比度 + 时间变化 + 颜色突兀

**输出**:
- 向丘脑枕: 空间显著性 → Pulvinar → 皮层
- 向脑干: 眼动指令

### 3.11 丘脑枕 (Pulvinar) — 第二条通路中继

**文件**: `cerebrum/thalamus/pulvinar.py` (新建)

**职责**: 上丘→皮层中继, 低空间频率快速通路

**前馈**: 从上丘接收空间显著性

**输出**: 向 V2/V4/MT 发送快速空间信号, 含低空间频率的"场景大局"信息

### 3.12 FPN 与跨通路绑定

**文件**: `cerebrum/association/fpn.py` (增强) + `cerebrum/association/visual_binding.py` (新建)

**职责**: 注意力探照灯调制各通道增益, 驱动跨通道特征绑定

**FPN 增益调制**:
```
对每个视觉区:
  gain_M = 1.0 + attention_weight × task_M_mask    (M 通道增益)
  gain_P = 1.0 + attention_weight × task_P_mask    (P 通道增益)
  gain_K = 1.0 + attention_weight × task_K_mask    (K 通道增益)
```

**绑定机制** (visual_binding.py):
- FPN 关注的空间位置 → 该位置上 M/P/K 通道的增益同时提升
- "绑定" = 同一空间位置 × 同一时间的 M/P/K 特征被联合增强
- 这与注意介导的绑定假说一致 (Treisman 的特征整合理论的神经实现)


### 4.3 设计原则

- 每个维度有两个标签: **通路** (M/P/K) + **脑区** (V1/V2/V4/MT/IT)
- 通路内: 脑区按层级排列 (低→高)
- 通路间: 保持间隔, 便于 FPN 按通道独立调制增益
- IT 向量是 Hebb 簇激活的紧凑编码, 不是分类标签

## 5. 预测编码嵌入

### 5.1 每步处理流程

```
每一步 agent.step() 中:

Phase 1 — 前馈 (自下而上, ~模拟一步):
  retinal_output = Retina.encode(image)           # M/P/K 分型
  lgn_output = LGN.relay(retinal_output,          # 6层门控中继
                          brainstem_state,
                          v1_feedback_prev)
  v1_output = V1.feedforward(lgn_output)          # 按层编码
  v2_output = V2.feedforward(v1_output)           # 按条纹编码
  mt_output = MT.feedforward(v1_M=v1_output.M,    # 背侧前馈
                             v2_thick=v2_output.thick)
  v4_output = V4.feedforward(v2_pale=v2_output.pale,
                             v2_thin=v2_output.thin)
  mst_output = MST.feedforward(mt_output)
  it_output = IT.feedforward(v4_output)

Phase 2 — 反馈 (自上而下, 携带预测):
  v4_pred = IT.feedback()                         # IT 预测 V4 该看到什么
  v2_pred = V4.feedback(v4_pred)                  # V4 预测 V2 该看到什么
  v1_pred = V2.feedback(v2_pred)                  # V2 预测 V1 该看到什么
  mt_pred = MST.feedback()                        # MST 预测 MT (运动预期)
  v1_mt_pred = MT.feedback(mt_pred)               # MT 预测 V1-4B (运动)

Phase 3 — 预测误差计算:
  PE_lgn = LGN.prediction_error()
  PE_v1 = V1.prediction_error(v1_pred, v1_mt_pred)
  PE_v2 = V2.prediction_error(v2_pred)
  PE_v4 = V4.prediction_error(v4_pred)
  PE_mt = MT.prediction_error(mt_pred)
  PE_mst = MST.prediction_error()
  PE_it = IT.prediction_error()

Phase 4 — 汇入自由能:
  F_accuracy = aggregate_PE(PE_v1, PE_v2, PE_v4, PE_mt, PE_mst, PE_it)
  # F_accuracy 汇入 Cingulate.compute_F() 的总自由能

Phase 5 — FPN 增益调制:
  FPN.gate_attention() → 按任务目标调制各通路的 gain

Phase 6 — 局部 Hebb 更新:
  for area in [V1, V2, V4, MT, MST, IT]:
      area.hebb_update()
```

### 5.2 预测误差聚合

```
F_accuracy = w_v1 × Σ|PE_v1|² / σ²_v1
           + w_v2 × Σ|PE_v2|² / σ²_v2
           + w_v4 × Σ|PE_v4|² / σ²_v4
           + w_mt × Σ|PE_mt|² / σ²_mt
           + w_it × Σ|PE_it|² / σ²_it

其中:
  σ²_layer: 该层的历史 PE 方差 → 自适应精度 (逆方差加权)
  w_layer:   层级权重 (高层 PE 在自由能中占比更大, 因更影响行为)
```

## 6. 格式塔效应的涌现路径

### 6.1 闭合律 (Closure) — 新实现

**生物机制**: IT→V4→V2→V1 反馈链

```
流程:
  1. V1 检测到不连续的边缘片段 (缺口处无响应)
  2. V2 的苍白条纹通过共线促进, 在缺口处产生弱的"虚拟边缘"信号
  3. V4 整合这些边缘 (包括虚拟边缘) → 检测到"近似闭合的轮廓"
  4. IT 的物体 Hebb 簇被部分激活 → "这可能是物体 X"
  5. IT 向下预测: "如果这是 X, V4 应该看到完整封闭轮廓"
  6. V4 向下预测: "V2 应该在缺口位置检测到边缘"
  7. V2 下调缺口位置的预测 → V1 在缺口处的 PE 降低

结果: 即使物理输入中不存在闭合边缘, PE 仍然低 → "感知到闭合"
```

**涌现特性**: 不需要显式检测"闭合", 闭合感 = IT 物体预测成功解释了 V1 的缺失信号

### 6.2 共同命运律 (Common Fate) — 新实现

**生物机制**: MT→V2 粗条纹→V2 苍白条纹的跨条纹连接

```
流程:
  1. MT 检测到一组点具有相同的运动方向和速度
  2. MST 整合这些点 → "这是一个运动组"
  3. MST→MT 反馈增强同向点的运动能量
  4. MT→V2 粗条纹反馈: "这些空间位置的点同向运动"
  5. V2 粗条纹→苍白条纹横向连接: 将运动信息传递给形状分析系统
  6. V2 苍白条纹提升这些点的空间分组权重

结果: 一起运动的点被 V2 的 P 通路判定为"属于同一组"
```

**涌现特性**: 不需要显式计算"共同命运"特征, 分组信号从 MT→V2 的跨通路连接中自然产生

### 6.3 简洁律 (Prägnanz) — 新实现

**生物机制**: F_cognitive 对模型复杂度的自然偏好

```
机制:
  - 多个可能的物体假设在 IT 中并行激活 (多个 Hebb 簇竞争)
  - 简单解释 → 占用少数 Hebb 簇 → 突触能耗低
  - 复杂解释 → 需要更多 Hebb 簇 + 更多层间迭代 → F_cognitive 高
  - 前额叶的 EFE 选择自然偏向 F_total 更低的解释

结果: 简洁的解释自动胜出, 不需要显式"选择简洁的"
```

**涌现特性**: 简洁律 = F_cognitive 最小化的自然产品, 不是算法规则

### 6.4 已有格式塔定律的迁移

| 格式塔定律 | v4.4 实现 (gestalt.py) | v5.0 实现 | 变化 |
|-----------|----------------------|-----------|------|
| **接近律** | KDTree 连通分量聚类 | 保留在 V2 苍白条纹, 作为空间临近度的自然检测 | 从独立特征 → V2 内部计算 |
| **相似律** | 跨朝向+跨尺度相关 | P/K 通道对同一空间位置的自上而下共调 → IT 的同一预测解释多个特征 | 从手工相关 → 预测驱动的共调 |
| **连续律** | 沿朝向轴追踪轮廓 | 保留在 V2 苍白条纹中, 作为共线促进的水平连接权重 | 本质不变, 归入 V2 |
| **对称律** | 镜像余弦相似度 | 保留在 V2/V4, 作为空间相关检测 | 本质不变 |
| **图底律** | 边缘密度 Otsu 分割 | V2 边界归属 + V4 表面 + FPN 注意偏向前景 → 边界内侧与 FPN 关注位置的匹配 | 从 Otsu 阈值 → FPN 驱动的竞争 |
| **闭合律** | *缺失* | IT→V4→V2→V1 反馈链 | **新增, 涌现** |
| **共同命运律** | *缺失* | MT→V2 跨条纹连接 | **新增, 涌现** |
| **简洁律** | *缺失* | F_cognitive 自然偏好 | **新增, 涌现** |

### 6.5 gestalt.py 的命运

- 接近律/连续律/对称律: 功能保留, 移入 V2 模块内部作为局部计算
- 相似律: 被 IT 预测驱动的共调取代
- 图底律: 被 FPN 驱动的竞争取代
- 闭合/共同命运/简洁: 从层级动力学中涌现

**最终目标**: `gestalt.py` 可以废弃。格式塔分组不再是独立的特征向量, 而是并行通路架构 + 预测编码的自然副产物。

## 10. 风险与缓解

| 风险 | 概率 | 影响 | 缓解 |
|------|------|------|------|
| 感知维度膨胀 (D 增长过多) | 中 | 下游模块 (海马/ACC/前额叶) 需重新适配 | 通过 PCA 或紧凑编码控制各通道维度, 目标 D<400 |
| 涌现效应不明显 (闭合律等未出现) | 中 | 格式塔效应仍需手工计算 | 预留回退路径: 若层级反馈未产生涌现, 可在 V2/V4 中保留简化版手工检测 |
| 计算成本显著增加 (6 个新模块 × 每步处理) | 低 | 交互式对话变慢 | 每个模块的计算是轻量的 (Gabor 卷积核共享, 反馈是矩阵乘, PE 是向量差) |
| 通路分离后通道间交互不足 | 中 | 感知过于碎片化 | V4 和 IT 设计为汇合点, FPN 绑定信号跨通道协调 |
| agent.step() 中的数据流过于复杂 | 低 | 调试和维护困难 | 统一的 VisualArea 接口模式 + 视觉管线编排函数 `visual_hierarchy.py` |

## 11. 与项目方法论承诺的契合

| 方法 | v5.0 如何遵守 |
|------|-------------|
| **仿脑区结构** | M/P/K 通路分离、LGN 6 层门控、V1 层状组织、V2 条纹 (粗/苍白/细)、MT/MST/IT/Pulvinar/SC 的位置和连接都直接映射自中枢视觉通路文档的解剖学描述 |
| **无现象学** | 格式塔定律不是手工规则, 而是通路架构 + 预测编码的副产品。闭合感 = PE 降低。共同命运 = MT→V2 反馈。简洁律 = F_cognitive 的自然偏好。不预设任何感知体验 |
| **无 LLM** | 所有视觉处理基于 Gabor 滤波 + Hebb 学习 + 预测编码。IT 的物体识别是 Hebb 簇的统计共现学习, 不依赖任何预训练深度网络 |
| **涌现优于工程** | 三个新格式塔定律 (闭合/共同命运/简洁) 的目标是从架构动力学中自然产生, 不写显式检测代码 |
| **身体是基础** | 视觉处理通过脑干状态 (唤醒度) 调制 LGN 门控 → 身体状态直接影响视觉信息流 |
| **记忆是核心** | IT 物体识别依赖 Hebb 簇 → 视觉知识从感知统计中自组织 |
| **自由能是唯一目标** | 所有层级的 PE 汇入 F_accuracy → 驱动行动选择和学习 → 预测误差最小化是统一框架 |
