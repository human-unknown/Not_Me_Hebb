Module B 详细设计: Divisive Normalization 亮度/颜色恒常性

修改文件: layer0_visual.py (GaborFilterBank 类)

新增内容

新增方法: _divisive_normalize(response_map, filter_idx) → normalized_map
输入: 单个 Gabor 滤波器的响应图 (64×64)
输出: 归一化后的响应图

计算步骤:
1. 构建 surround 高斯核 (σ=image_size/16, radius=image_size/16)
2. 对响应取绝对值 → 用 surround 核卷积 → 得到局部能量估计
3. normalized = response / (semi_saturation² + local_energy)
4. 对颜色通道: 额外做跨通道归一化 (RG 和 BY 互除)

参数 (硬编码, 不加入 Theta):
semi_saturation = 0.1 (仿 V1 的 σ² 半饱和常数)
surround_sigma = image_size / 16 (≈4 像素 for 64×64)
surround_radius = image_size // 16 (≈4 像素)
修改方法: encode(), encode_v2(), encode_v4(), encode_color()
在现有管线中的插入点:

encode():
for i in range(n_filters):
response = ifft2(image_fft * kernel_ffts[i]) ← 现有
response = _divisive_normalize(response, i) ← 新增
modulated = response * gains[i] ← 现有

encode_color():
for each opponent channel (RG, BY):
... Gabor 卷积 ...
response = _divisive_normalize(response, i) ← 新增
跨通道归一化: RG_norm /= (1 + BY_local_energy) ← 新增
预计算 (__init__ 中): surround 卷积核
# 在 __init__ 中新增:
surround_size = self.image_size // 16 * 2 + 1 # ≈9
self._surround_kernel = gaussian_2d(sigma=image_size/16, size=surround_size)
self._surround_kernel_fft = fft2(pad_to_fft_size(surround_kernel)) # 预计算 FFT

# _divisive_normalize 中用 FFT 卷积 (与现有 Gabor 卷积一致):
local_energy = |ifft2(fft2(|response|) * surround_kernel_fft)|
normed = response / (semi_sat² + local_energy)
